# BetYV-Analyzer
Sistema de Estimación Probabilística y Análisis de Riesgo en Apuestas Deportivas 

Es un sistema de análisis y modelado probabilístico orientado a la evaluación cuantitativa de apuestas deportivas combinadas (parleys). El proyecto tiene como objetivo estimar probabilidades reales de eventos deportivos utilizando datos históricos y técnicas de modelado estadístico, con el fin de analizar el riesgo y el valor esperado asociado a este tipo de decisiones bajo incertidumbre.

En una fase inicial el sistema se enfoca en un deporte específico (por ejemplo, fútbol) y emplea variables como:

-Resultados históricos

-Goles anotados y recibidos

-Rendimiento como local y visitante

-Estadísticas agregadas relevantes

A partir de estos datos, se implementan modelos predictivos como:

-Regresión logística

-Modelo de Poisson para estimación de goles

-Simulación Monte Carlo para análisis de riesgo
